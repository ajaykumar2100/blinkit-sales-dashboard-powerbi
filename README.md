# 🛒 Blinkit Sales Dashboard (Power BI)

## 📊 Project Overview

This project analyzes Blinkit sales data to uncover insights about outlet performance, product categories, and customer preferences.

## 🎯 Key Insights

* Tier 3 outlets generated highest sales
* Peak sales observed in 2018
* Fruits & Snack Foods are top-performing categories

## 📌 Features

* Interactive dashboard with slicers
* KPI cards (Sales, Avg Sales, Items, Rating)
* Sales trend analysis
* Outlet and item-level insights

## 🛠 Tools Used

* Power BI
* Excel / CSV
* Data Cleaning & Transformation
